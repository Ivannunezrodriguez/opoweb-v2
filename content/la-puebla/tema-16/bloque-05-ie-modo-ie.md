# Tema 16 · Bloque 5 · Internet Explorer 11 y modo Internet Explorer

**Estado:** APROBADO POR EL USUARIO

## 1. Situación actual de Internet Explorer

Internet Explorer 11 fue la última versión principal del navegador Internet Explorer.

Microsoft documenta que:

- la aplicación de escritorio IE11 terminó el soporte para determinadas versiones de Windows 10 el 15 de junio de 2022;
- fue deshabilitada permanentemente en esas versiones mediante una actualización de Microsoft Edge;
- la recomendación actual es utilizar Microsoft Edge;
- la compatibilidad con aplicaciones antiguas se realiza mediante el modo Internet Explorer de Edge.

Por tanto, Internet Explorer se estudia en este tema por:

- aparecer literalmente en el programa oficial;
- mantener valor histórico y conceptual en preguntas de ofimática;
- existir aplicaciones heredadas que requieren su motor;
- ser necesario distinguirlo de Edge y del modo IE.

No debe presentarse como navegador ordinario seguro y vigente para uso general.

## 2. Interfaz clásica de Internet Explorer 11

Elementos tradicionales:

- barra de direcciones;
- pestañas;
- botones Atrás, Adelante, Actualizar y Detener;
- botón Inicio;
- Centro de favoritos;
- menú Herramientas;
- barra de comandos o menú clásico, según configuración;
- administrador de descargas;
- opciones de Internet.

La apariencia podía variar por Windows y personalización.

## 3. Funciones clásicas aún preguntables

### Navegación

- escribir dirección;
- seguir enlaces;
- abrir pestañas y ventanas;
- volver atrás o adelante;
- actualizar;
- buscar dentro de la página;
- usar zoom;
- imprimir.

### Favoritos, fuentes e historial

El Centro de favoritos agrupaba:

- Favoritos;
- Fuentes RSS;
- Historial.

Esto es una diferencia visible respecto de la organización actual de Edge.

### Descargas

IE11 disponía de administrador de descargas para ver, abrir, pausar o cancelar archivos.

### InPrivate

En Internet Explorer 11, `Ctrl + Mayús + P` abría una ventana o pestaña InPrivate.

En Edge actual, la ventana InPrivate se abre con `Ctrl + Mayús + N`. El mismo atajo no debe trasladarse de un navegador a otro.

## 4. Atajos clásicos relevantes de IE11

| Atajo | Acción tradicional |
|---|---|
| `Ctrl + T` | nueva pestaña |
| `Ctrl + N` | nueva ventana |
| `Ctrl + W` | cerrar pestaña |
| `Ctrl + L` o `Alt + D` | seleccionar barra de direcciones |
| `Ctrl + F` | buscar en página |
| `Ctrl + D` | agregar a Favoritos |
| `Ctrl + H` | historial |
| `Ctrl + J` | descargas |
| `Ctrl + Mayús + P` | InPrivate |
| `F5` | actualizar |
| `F11` | pantalla completa |

Algunos atajos coinciden con Edge, pero otros no.

## 5. Opciones de Internet

Internet Explorer utilizaba el cuadro **Opciones de Internet**, también accesible históricamente desde Windows, para configurar:

- página principal;
- historial y archivos temporales;
- seguridad;
- privacidad;
- contenido;
- conexiones;
- programas;
- opciones avanzadas.

Muchas aplicaciones heredadas siguen haciendo referencia a estas opciones o a componentes del sistema, aunque la navegación moderna se configure desde Edge.

## 6. Zonas de seguridad de Internet Explorer

IE organizaba la seguridad mediante zonas:

- Internet;
- Intranet local;
- Sitios de confianza;
- Sitios restringidos.

Cada zona podía aplicar niveles y reglas diferentes.

No debe confundirse:

- agregar un sitio a **Sitios de confianza**;
- permitirlo en SmartScreen;
- configurarlo para modo IE;
- conceder permisos de cámara o notificaciones en Edge.

Son mecanismos distintos.

## 7. Vista de compatibilidad

Internet Explorer incluía funciones de compatibilidad para páginas diseñadas para versiones antiguas.

La **Vista de compatibilidad** clásica no es idéntica al **modo Internet Explorer de Edge**:

- Vista de compatibilidad actuaba dentro de Internet Explorer;
- modo IE funciona dentro de Edge y utiliza el motor heredado para sitios configurados.

## 8. ¿Qué es el modo Internet Explorer?

El modo IE de Microsoft Edge permite abrir aplicaciones y sitios heredados que requieren tecnologías de Internet Explorer.

Microsoft explica que Edge utiliza:

- el motor Chromium para sitios modernos;
- el motor Trident/MSHTML de IE11 para sitios configurados en modo IE.

Cuando una página se carga en modo IE aparece un indicador con el logotipo de Internet Explorer en la barra de navegación.

## 9. Qué no significa el modo IE

No significa:

- que Internet Explorer clásico vuelva a estar instalado como navegador general;
- que todos los sitios se abran con el motor antiguo;
- que el usuario pueda decidir libremente cualquier sitio en todos los equipos;
- que todas las barras, complementos o herramientas de IE sean compatibles;
- que las extensiones modernas de Edge puedan interactuar siempre con el contenido heredado.

## 10. Configuración organizativa

En entornos empresariales o administrativos, los sitios de modo IE suelen definirse mediante políticas y una lista de sitios empresariales.

La organización decide:

- qué URL requieren modo IE;
- cuánto tiempo permanece la configuración;
- si el usuario puede recargar manualmente una página en modo IE;
- qué políticas y controles se aplican.

Esto evita abrir todo Internet con un motor heredado.

## 11. Compatibilidad del modo IE

Microsoft se compromete a mantener el modo IE al menos hasta 2029 en sistemas operativos compatibles y a avisar con un mínimo de un año antes de su retirada.

La fecha se refiere al soporte del **modo IE**, no a la aplicación de escritorio Internet Explorer 11.

## 12. Limitaciones

Microsoft documenta limitaciones como:

- no se admiten las barras de herramientas clásicas de Internet Explorer;
- determinadas configuraciones de navegación de IE no se aplican igual;
- las herramientas de desarrollo clásicas no funcionan del mismo modo;
- las extensiones de Edge que interactúan directamente con el contenido pueden no funcionar en páginas IE.

Además, el modo IE depende de actualizaciones y políticas correctas.

## 13. Cuándo usarlo

Solo debe utilizarse cuando:

- una aplicación heredada lo requiere;
- la organización lo ha configurado o autorizado;
- existe una necesidad de compatibilidad documentada.

Para páginas modernas, se utiliza el modo normal de Edge.

## 14. Comparación resumida

| Elemento | Internet Explorer 11 | Edge | Modo IE en Edge |
|---|---|---|---|
| Estado | retirado en gran parte del escritorio Windows | navegador actual | función de compatibilidad |
| Motor | Trident/MSHTML | Chromium | Trident/MSHTML dentro de Edge |
| Uso | heredado | navegación general | aplicaciones antiguas concretas |
| Configuración | Opciones de Internet | Configuración de Edge | políticas y lista de sitios |
| InPrivate | `Ctrl + Mayús + P` | `Ctrl + Mayús + N` | depende de Edge y políticas |
| Extensiones | complementos heredados | extensiones actuales | compatibilidad limitada |

## 15. Migración de datos

Edge puede importar desde Internet Explorer datos compatibles como:

- favoritos;
- contraseñas;
- historial;
- determinadas configuraciones.

Importar copia datos al perfil de Edge. No reactiva Internet Explorer ni convierte los sitios antiguos en modernos.

## 16. Riesgos de una respuesta desactualizada

En examen pueden aparecer formulaciones antiguas. Debe responderse distinguiendo:

- función histórica de Internet Explorer;
- situación vigente del producto;
- equivalencia actual en Edge;
- modo IE como compatibilidad.

Ejemplo:

> “Internet Explorer es el navegador actual predeterminado de Windows”.

Es incorrecto en el contexto actual. Edge es la referencia vigente y Microsoft recomienda la migración.

## 17. Errores frecuentes

1. Decir que IE11 sigue siendo el navegador moderno recomendado.
2. Confundir final de soporte de IE11 con final de modo IE.
3. Confundir modo IE con la aplicación clásica.
4. Suponer que todos los sitios se abren con Trident.
5. Confundir Vista de compatibilidad y modo IE.
6. Confundir Sitios de confianza con lista de modo IE.
7. Trasladar `Ctrl + Mayús + P` de IE a Edge como atajo InPrivate.
8. Pensar que las barras de herramientas antiguas funcionan en modo IE.
9. Abrir páginas modernas en modo IE sin necesidad.
10. Creer que importar favoritos reactiva Internet Explorer.
