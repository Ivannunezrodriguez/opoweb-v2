# Tema 16 · Bloque 1 · Conceptos de Internet, web, navegador y navegación

**Estado:** APROBADO POR EL USUARIO

## 1. Internet, web y navegador

Internet es una red mundial de redes. La Web es un servicio que funciona sobre Internet mediante documentos y recursos enlazados. El navegador es la aplicación cliente que solicita, interpreta y muestra esos recursos.

Diferencias básicas:

| Concepto | Significado |
|---|---|
| Internet | infraestructura de comunicaciones |
| Web | servicio de páginas y recursos enlazados |
| Navegador | programa usado para acceder a la Web |
| Motor de búsqueda | servicio que indexa y localiza contenidos |
| Sitio web | conjunto de páginas relacionadas |
| Página web | recurso o documento concreto |

Microsoft Edge e Internet Explorer son navegadores. Bing, Google o DuckDuckGo son motores de búsqueda.

## 2. Elementos de una dirección web

Una URL puede contener:

- esquema o protocolo: `https`;
- nombre de host: `sede.ejemplo.es`;
- puerto, normalmente implícito;
- ruta: `/tramites/registro`;
- consulta: `?anio=2026`;
- fragmento: `#requisitos`.

Ejemplo completo:

`https://sede.ejemplo.es/tramites/registro?anio=2026#requisitos`

### Dominio y subdominio

En `sede.ejemplo.es`:

- `.es` es el dominio de nivel superior;
- `ejemplo.es` es el dominio registrable;
- `sede` es un subdominio.

La identidad debe comprobarse leyendo el dominio real, no solo el texto visible de un enlace.

## 3. Barra de direcciones y búsqueda

La barra de direcciones de Edge cumple varias funciones:

- introducir una URL;
- realizar una búsqueda;
- mostrar la dirección actual;
- indicar información de seguridad y permisos del sitio;
- ofrecer sugerencias basadas en configuración, historial o favoritos.

`Ctrl + L` selecciona la barra de direcciones.

No es igual:

- escribir `https://www.boe.es` y acceder directamente;
- escribir `BOE` y aceptar resultados de un buscador.

En el segundo caso puede aparecer publicidad, resultados no oficiales o páginas de terceros.

## 4. Hipervínculos

Un enlace permite abrir otro recurso. Puede apuntar a:

- otra página;
- una sección de la misma página;
- un archivo descargable;
- una dirección de correo;
- una aplicación o protocolo específico.

Antes de seguir un enlace sospechoso conviene comprobar su destino real, especialmente en correos y mensajes.

## 5. Navegación básica

### Atrás y adelante

- **Atrás:** regresa a la página anterior del historial de la pestaña.
- **Adelante:** vuelve a una página abandonada mediante Atrás.

No equivalen a cerrar o abrir pestañas.

### Recargar

`Ctrl + R` o `F5` vuelve a solicitar la página. Puede actualizar contenido y repetir algunas solicitudes; por eso debe actuarse con cautela tras enviar formularios o pagos.

### Detener

Interrumpe la carga en curso. No deshace datos que ya hayan sido enviados al servidor.

### Inicio y nueva pestaña

La página de inicio configurada y la página de nueva pestaña pueden ser distintas. El botón Inicio puede estar oculto o deshabilitado.

## 6. Pestañas, ventanas y procesos

Una ventana puede contener varias pestañas. Cada pestaña mantiene su propia navegación, pero comparte elementos del perfil como favoritos, configuración y ciertos datos de sesión.

Acciones comunes:

- `Ctrl + T`: nueva pestaña;
- `Ctrl + N`: nueva ventana;
- `Ctrl + W`: cerrar pestaña;
- `Ctrl + Mayús + T`: reabrir pestaña cerrada;
- `Ctrl + Tab`: pestaña siguiente;
- `Ctrl + Mayús + Tab`: pestaña anterior;
- `Ctrl + 1` a `Ctrl + 8`: ir a una pestaña concreta;
- `Ctrl + 9`: última pestaña.

Cerrar la última pestaña suele cerrar la ventana. Cerrar una ventana no borra automáticamente historial, cookies ni descargas.

## 7. Abrir enlaces

Un enlace puede abrirse:

- en la pestaña actual;
- en una nueva pestaña;
- en una nueva ventana;
- en una ventana InPrivate.

El menú contextual permite elegir. `Ctrl + clic` suele abrir en una pestaña nueva y el clic con la rueda del ratón suele tener el mismo efecto.

## 8. Búsqueda dentro de una página

`Ctrl + F` busca texto en la página cargada. No realiza una búsqueda general en Internet.

La búsqueda en página:

- localiza coincidencias visibles o interpretadas por el navegador;
- puede no encontrar texto contenido dentro de imágenes;
- puede variar en páginas que cargan contenido dinámicamente.

## 9. Zoom, pantalla completa e impresión

- `Ctrl + +`: aumentar zoom;
- `Ctrl + -`: reducir zoom;
- `Ctrl + 0`: restablecer zoom;
- `F11`: alternar pantalla completa;
- `Ctrl + P`: imprimir.

El zoom modifica la presentación, no necesariamente el tamaño real de imágenes o archivos descargados.

## 10. Estados de carga y errores

Errores frecuentes:

- dirección inexistente;
- servidor no disponible;
- falta de conexión;
- resolución DNS fallida;
- certificado no válido;
- acceso denegado;
- recurso movido o eliminado;
- tiempo de espera agotado.

Ante un error debe distinguirse si el problema está en:

1. la conexión del equipo;
2. la dirección escrita;
3. el navegador o su caché;
4. el servidor remoto;
5. la autenticación o permisos;
6. la seguridad del certificado.

## 11. Formularios web

Los formularios permiten introducir y enviar datos. Antes de enviarlos hay que comprobar:

- dominio;
- conexión segura cuando proceda;
- campos obligatorios;
- archivos adjuntos correctos;
- confirmación final;
- justificante o número de registro.

Recargar o usar Atrás después de enviar un formulario puede mostrar advertencias o provocar una repetición. No debe repetirse una operación administrativa o pago sin comprobar su estado.

## 12. Navegación accesible

El navegador permite:

- moverse con `Tab` y `Mayús + Tab`;
- activar el elemento enfocado con `Enter` o barra espaciadora según control;
- ampliar zoom;
- usar lectura en voz alta o lector inmersivo cuando estén disponibles;
- cambiar tamaño de texto y contraste desde sistema o navegador.

## 13. Errores de examen

1. Decir que Internet y Web son exactamente lo mismo.
2. Confundir navegador con buscador.
3. Confundir una URL con el texto visible del enlace.
4. Pensar que `Ctrl + F` busca en toda Internet.
5. Confundir recargar con volver atrás.
6. Suponer que cerrar pestaña borra datos de navegación.
7. Creer que pantalla completa aumenta la resolución del contenido.
8. Repetir el envío de un formulario sin verificar el resultado.
