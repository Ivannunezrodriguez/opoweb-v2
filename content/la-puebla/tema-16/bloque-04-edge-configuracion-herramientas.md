# Tema 16 · Bloque 4 · Microsoft Edge: configuración, perfiles y herramientas

**Estado:** EN REVISIÓN DEL USUARIO

## 1. Interfaz esencial de Edge

Elementos habituales:

- barra de pestañas;
- botones Atrás, Adelante y Recargar;
- barra de direcciones y búsqueda;
- información del sitio;
- botón Favoritos;
- perfil;
- menú **Configuración y más**;
- panel de descargas, historial o favoritos;
- barra lateral u otras funciones que pueden variar por versión.

El navegador se actualiza continuamente, por lo que debe estudiarse la función y no una posición visual inmutable.

## 2. Configuración y más

El menú de tres puntos permite acceder, entre otras opciones, a:

- nueva pestaña, ventana o ventana InPrivate;
- zoom;
- favoritos;
- historial;
- descargas;
- extensiones;
- imprimir;
- buscar en la página;
- configuración;
- ayuda y comentarios.

Muchas funciones también disponen de atajo de teclado.

## 3. Atajos principales de Edge

| Atajo | Acción |
|---|---|
| `Ctrl + T` | abrir pestaña nueva |
| `Ctrl + N` | abrir ventana nueva |
| `Ctrl + Mayús + N` | abrir ventana InPrivate |
| `Ctrl + W` | cerrar pestaña |
| `Ctrl + Mayús + T` | reabrir pestaña cerrada |
| `Ctrl + L` | seleccionar barra de direcciones |
| `Ctrl + R` o `F5` | recargar |
| `Ctrl + F` | buscar en la página |
| `Ctrl + D` | guardar como favorito |
| `Ctrl + Mayús + O` | abrir favoritos |
| `Ctrl + H` | abrir historial |
| `Ctrl + J` | abrir descargas |
| `Ctrl + P` | imprimir |
| `Ctrl + Mayús + Supr` | borrar datos de navegación |
| `F11` | pantalla completa |

Debe tenerse cuidado con los atajos heredados: `Ctrl + Mayús + P` abría InPrivate en Internet Explorer, pero en Edge se relaciona con la impresión mediante el cuadro del sistema.

## 4. Configuración de inicio, página principal y nueva pestaña

Son conceptos distintos:

- **Al iniciar Edge:** puede abrir una pestaña nueva, continuar la sesión anterior o cargar páginas concretas.
- **Botón Inicio:** puede configurarse para abrir una página determinada o la nueva pestaña.
- **Página de nueva pestaña:** contenido mostrado al crear una pestaña.

Modificar una opción no cambia necesariamente las otras.

## 5. Navegador predeterminado

El navegador predeterminado es el utilizado por Windows para abrir enlaces web asociados cuando no se elige otro programa.

Establecer Edge como predeterminado:

- no impide abrir manualmente otro navegador;
- no cambia el motor de búsqueda por sí solo;
- no activa automáticamente el modo IE para todos los sitios.

## 6. Motor de búsqueda

La barra de direcciones puede usar un motor de búsqueda predeterminado. Cambiarlo afecta a búsquedas, no a las URL escritas de forma completa.

Debe distinguirse:

- navegador predeterminado del sistema;
- motor de búsqueda predeterminado del navegador;
- página de inicio;
- página de nueva pestaña.

## 7. Perfiles

Un perfil mantiene separados datos y configuración, como:

- favoritos;
- historial;
- contraseñas;
- extensiones;
- datos de formularios;
- cookies y sesiones;
- configuración de sincronización.

Pueden existir perfiles personal, laboral o invitado.

### Cambiar de perfil

Cambiar de perfil puede cambiar:

- cuenta autenticada;
- favoritos disponibles;
- sesiones web;
- historial;
- políticas de la organización;
- extensiones permitidas.

No debe usarse el perfil personal para datos institucionales si la organización exige uno administrado.

## 8. Inicio de sesión y sincronización

Al iniciar sesión y activar sincronización, Edge puede sincronizar:

- favoritos;
- contraseñas;
- historial;
- extensiones;
- configuración;
- pestañas abiertas;
- datos de autorrelleno;
- información de pago, según opciones.

La sincronización puede configurarse por categorías.

Diferencias:

- cerrar una pestaña no cierra la sesión del perfil;
- cerrar sesión del perfil no borra automáticamente todos los datos locales;
- eliminar el perfil puede retirar sus datos del dispositivo;
- borrar datos sincronizados puede tener efectos en otros dispositivos.

## 9. Importar datos

Edge puede importar desde otros navegadores:

- favoritos;
- contraseñas;
- historial;
- pestañas;
- configuración compatible.

La importación copia datos; no obliga a desinstalar el navegador anterior.

Después de importar deben revisarse duplicados y credenciales no deseadas.

## 10. Extensiones

Las extensiones añaden funciones al navegador.

Antes de instalar:

1. comprobar editor y procedencia;
2. revisar permisos;
3. valorar necesidad;
4. evitar extensiones duplicadas;
5. mantenerlas actualizadas;
6. desinstalarlas si dejan de ser necesarias.

Permisos sensibles incluyen leer y modificar datos de páginas, historial o descargas.

En modo InPrivate una extensión solo funciona si se le concede permiso específico.

## 11. Lector PDF

Edge puede abrir archivos PDF dentro del navegador y permite, según documento y versión:

- navegar por páginas;
- buscar texto;
- ampliar o reducir;
- imprimir;
- descargar;
- rellenar formularios compatibles;
- anotar o resaltar en determinadas funciones.

Abrir un PDF en Edge no implica que el documento esté firmado, sea auténtico o pueda editarse como un procesador de texto.

## 12. Impresión desde Edge

`Ctrl + P` abre la vista de impresión de Edge. Puede configurarse:

- impresora;
- páginas;
- copias;
- orientación;
- escala;
- márgenes;
- encabezados y pies;
- impresión a PDF.

`Ctrl + Mayús + P` puede abrir el cuadro de impresión del sistema en Edge. No debe confundirse con el atajo histórico de InPrivate de Internet Explorer.

## 13. Descargas y ubicación

Edge permite configurar:

- carpeta predeterminada;
- preguntar dónde guardar cada archivo;
- comportamiento tras finalizar;
- apertura de determinados tipos, sujeto a seguridad y políticas.

Cambiar la carpeta predeterminada no mueve los archivos ya descargados.

## 14. Idiomas, traducción y accesibilidad

Funciones posibles:

- idiomas preferidos;
- revisión ortográfica;
- traducción de páginas;
- zoom;
- lectura en voz alta;
- lector inmersivo en páginas compatibles;
- navegación por teclado.

La traducción automática puede contener errores y no sustituye una traducción oficial.

## 15. Configuración administrada

En equipos corporativos o públicos, Edge puede estar administrado mediante políticas. Esto puede limitar:

- navegador predeterminado;
- extensiones;
- InPrivate;
- sincronización;
- descargas;
- páginas de inicio;
- permisos;
- modo IE.

La indicación **Administrado por tu organización** significa que determinadas opciones proceden de políticas, no necesariamente de un fallo.

## 16. Actualizaciones y versión

Edge se actualiza con frecuencia. La versión puede consultarse en **Ayuda y comentarios > Acerca de Microsoft Edge**.

Actualizar Edge puede:

- corregir vulnerabilidades;
- cambiar menús;
- incorporar funciones;
- retirar comportamientos antiguos;
- requerir reinicio del navegador.

## 17. Restablecer configuración

Restablecer puede devolver opciones a valores predeterminados, pero no debe utilizarse sin valorar:

- extensiones;
- página de inicio;
- motor de búsqueda;
- pestañas fijadas;
- cookies y sesiones;
- datos sincronizados.

No equivale necesariamente a desinstalar ni borrar todos los datos.

## 18. Errores frecuentes

1. Confundir navegador predeterminado con motor de búsqueda.
2. Confundir página de inicio con nueva pestaña.
3. Pensar que importar elimina los datos del navegador anterior.
4. Confundir cerrar sesión con eliminar perfil.
5. Instalar extensiones sin revisar permisos.
6. Creer que abrir un PDF en Edge acredita su autenticidad.
7. Confundir `Ctrl + Mayús + P` de Edge con InPrivate.
8. Cambiar carpeta de descargas esperando mover archivos antiguos.
9. Interpretar una política corporativa como error del navegador.
10. Memorizar una ubicación exacta de menú como permanente.
