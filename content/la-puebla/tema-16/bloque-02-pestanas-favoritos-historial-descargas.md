# Tema 16 · Bloque 2 · Pestañas, favoritos, historial, búsqueda y descargas

**Estado:** EN REVISIÓN DEL USUARIO

## 1. Gestión de pestañas

Las pestañas permiten mantener varias páginas abiertas en una misma ventana.

Operaciones habituales:

- crear una pestaña;
- duplicarla;
- fijarla;
- silenciarla;
- moverla dentro de la ventana;
- trasladarla a otra ventana;
- agruparla cuando la versión del navegador lo permita;
- cerrarla;
- reabrirla.

Atajos principales de Edge:

| Atajo | Acción |
|---|---|
| `Ctrl + T` | nueva pestaña |
| `Ctrl + W` | cerrar pestaña |
| `Ctrl + Mayús + T` | reabrir pestaña cerrada |
| `Ctrl + Tab` | pestaña siguiente |
| `Ctrl + Mayús + Tab` | pestaña anterior |
| `Ctrl + Mayús + K` | duplicar pestaña |

Una pestaña fijada ocupa menos espacio y suele conservar únicamente el icono. Fijarla no la convierte en página de inicio ni impide cerrarla.

## 2. Ventanas normales e InPrivate

- `Ctrl + N`: ventana normal nueva.
- `Ctrl + Mayús + N`: ventana InPrivate en Edge.

Las pestañas InPrivate de una misma ventana comparten la sesión privada. Al cerrar todas las ventanas InPrivate se eliminan los datos temporales definidos por Edge para esa sesión, pero los archivos descargados y favoritos creados permanecen.

## 3. Favoritos

Un favorito conserva una dirección para acceder posteriormente.

### Crear

Puede hacerse mediante:

- icono de estrella de la barra de direcciones;
- `Ctrl + D`;
- menú **Configuración y más > Favoritos**.

Al guardar se puede:

- cambiar el nombre;
- elegir carpeta;
- crear una carpeta nueva.

### Organizar

Los favoritos admiten:

- carpetas y subcarpetas;
- cambio de nombre;
- traslado;
- eliminación;
- importación y exportación;
- visualización en la barra de favoritos.

`Ctrl + Mayús + O` abre la administración de favoritos en Edge.

### Favorito frente a acceso directo

- El favorito se administra dentro del navegador.
- Un acceso directo de Internet es un archivo o vínculo del sistema que apunta a una URL.
- Ninguno garantiza que el contenido siga disponible si el sitio cambia o desaparece.

### Favorito frente a página guardada

Guardar como favorito almacena la referencia. Guardar una página o imprimirla a PDF crea un archivo local, aunque ese archivo puede no reproducir todas las funciones dinámicas del sitio.

## 4. Barra de favoritos

Puede configurarse para mostrarse:

- siempre;
- nunca;
- solo en nuevas pestañas.

Mostrar la barra no cambia la existencia de los favoritos. Solo modifica su presentación.

## 5. Historial de navegación

El historial registra normalmente:

- URL visitada;
- título;
- fecha y hora;
- relación temporal con otras visitas.

`Ctrl + H` abre el historial en Edge.

Usos:

- volver a una página visitada;
- localizar una visita por fecha o búsqueda;
- reabrir pestañas o ventanas cerradas;
- eliminar entradas concretas o intervalos.

## 6. Eliminación de datos de navegación

`Ctrl + Mayús + Supr` abre normalmente el cuadro para borrar datos.

Categorías habituales:

- historial de exploración;
- historial de descargas;
- cookies y datos de sitios;
- imágenes y archivos en caché;
- contraseñas;
- datos de autorrelleno;
- permisos de sitios.

Debe elegirse el intervalo temporal correcto. Borrar datos sincronizados puede afectar a otros dispositivos según la configuración de la cuenta.

### Consecuencias

| Dato borrado | Consecuencia principal |
|---|---|
| Historial | desaparece la lista de páginas visitadas |
| Historial de descargas | desaparece la lista, no los archivos |
| Cookies | pueden cerrarse sesiones y perderse preferencias |
| Caché | los recursos deberán descargarse de nuevo |
| Contraseñas | se eliminan credenciales guardadas seleccionadas |
| Autorrelleno | se eliminan datos usados para completar formularios |
| Permisos | los sitios deberán volver a solicitarlos |

## 7. Descargas

`Ctrl + J` abre Descargas.

Una descarga puede tener estados como:

- en curso;
- pausada;
- completada;
- cancelada;
- bloqueada;
- fallida.

Acciones posibles:

- abrir archivo;
- abrir carpeta contenedora;
- pausar o reanudar;
- cancelar;
- eliminar de la lista;
- borrar el archivo desde el sistema.

Eliminar una entrada de la lista no equivale a borrar el archivo.

## 8. Guardar y abrir archivos

Al descargar o guardar debe comprobarse:

1. nombre;
2. extensión;
3. carpeta de destino;
4. existencia de un archivo anterior con el mismo nombre;
5. procedencia;
6. advertencias del navegador o antivirus.

La opción **Abrir** puede utilizar un archivo temporal o abrirlo tras descargarlo. Para conservarlo de forma controlada es preferible conocer su ubicación definitiva.

## 9. Riesgos de las descargas

Se debe desconfiar de:

- ejecutables no solicitados;
- extensiones dobles o nombres engañosos;
- archivos que exigen desactivar seguridad;
- documentos con macros inesperadas;
- páginas que imitan portales oficiales;
- descargas desde anuncios o enlaces acortados sin verificar.

La extensión visible puede ocultarse si Windows está configurado para ello. La identidad del archivo no debe deducirse únicamente del icono.

## 10. Búsqueda desde la barra de direcciones

Edge combina dirección y búsqueda. El motor predeterminado puede configurarse.

Diferencias:

- búsqueda general: localiza páginas indexadas;
- búsqueda en página (`Ctrl + F`): localiza texto en la página actual;
- búsqueda en historial o favoritos: filtra los datos guardados por el navegador;
- búsqueda de Windows: localiza aplicaciones, archivos y resultados configurados en el sistema.

## 11. Imprimir y guardar como PDF

`Ctrl + P` abre el cuadro de impresión. Puede elegirse una impresora física o una impresora virtual como **Microsoft Print to PDF**.

Guardar una página como PDF:

- crea una representación estática;
- puede cambiar diseño y paginación;
- no conserva todas las funciones interactivas;
- no equivale a guardar un favorito;
- no acredita por sí solo autenticidad o vigencia.

## 12. Reapertura de sesiones

Edge puede ofrecer:

- abrir la página de nueva pestaña;
- continuar donde se dejó;
- abrir páginas específicas.

Recuperar pestañas cerradas mediante historial no garantiza restaurar formularios no enviados, sesiones caducadas o contenido dinámico.

## 13. Sincronización

Con inicio de sesión y sincronización activa pueden sincronizarse:

- favoritos;
- historial;
- pestañas abiertas;
- contraseñas;
- extensiones;
- configuración;
- datos de autorrelleno.

Sincronizar no es realizar una copia de seguridad completa ni significa que todos los datos sean públicos. Debe distinguirse dato local, dato sincronizado y dato almacenado por el sitio web.

## 14. Errores frecuentes

1. Confundir pestaña con ventana.
2. Creer que fijar una pestaña la hace permanente.
3. Confundir favorito con copia local.
4. Pensar que eliminar historial de descargas borra archivos.
5. Borrar cookies sin prever cierre de sesiones.
6. Abrir un archivo bloqueado sin verificar origen.
7. Confundir búsqueda general con búsqueda en página.
8. Considerar un PDF guardado como prueba automática de autenticidad.
9. Pensar que sincronización equivale a copia de seguridad total.
10. Reabrir una pestaña esperando recuperar datos de formulario no enviados.
