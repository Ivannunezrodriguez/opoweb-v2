# Tema 38 · LibreOffice 24 IV. Elaboración y uso de presentaciones con LibreOffice 24 Impress

## 1. Concepto y finalidad

**LibreOffice Impress** es la aplicación de LibreOffice destinada a crear y mostrar presentaciones de diapositivas. Permite combinar texto, imágenes, tablas, gráficos, audio, vídeo, objetos de dibujo, animaciones y transiciones.

El formato nativo de las presentaciones de Impress es **ODP**.

> ⚠️ **¡Foco Examen!:** El formato nativo de Impress es **ODP**; PPT y PPTX son formatos compatibles, pero no son el formato nativo.

## 2. Interfaz principal

La ventana de Impress incluye, entre otros elementos:

- barra de título;
- menús y barras de herramientas;
- panel de diapositivas;
- área de trabajo;
- barra lateral;
- barra de estado.

### Vistas principales

| Vista | Finalidad |
|---|---|
| Normal | Crear y editar una diapositiva |
| Esquema | Trabajar con la estructura textual |
| Notas | Añadir notas para el presentador |
| Clasificador de diapositivas | Reordenar, copiar o eliminar diapositivas |
| Patrón | Definir elementos comunes |

> ⚠️ **¡Foco Examen!:** El **Clasificador de diapositivas** facilita la reorganización global; la vista **Normal** se utiliza para editar el contenido de una diapositiva.

## 3. Crear, abrir y guardar presentaciones

Una presentación nueva puede crearse desde **Archivo > Nuevo > Presentación** o con **Ctrl + N** cuando Impress está activo.

Operaciones esenciales:

| Acción | Atajo |
|---|---:|
| Guardar | **Ctrl + S** |
| Guardar como | **Ctrl + Mayús + S** |
| Abrir | **Ctrl + O** |
| Imprimir | **Ctrl + P** |
| Deshacer | **Ctrl + Z** |
| Rehacer | **Ctrl + Y** |

Al guardar en formatos ajenos, Impress puede advertir de posibles pérdidas de formato o funcionalidad.

> ⚠️ **¡Foco Examen!:** Guardar como PPTX no convierte PPTX en formato nativo ni garantiza conservar todas las funciones propias de Impress.

## 4. Diapositivas y diseños

Cada presentación se compone de diapositivas. Sobre ellas pueden realizarse operaciones como:

- insertar;
- duplicar;
- eliminar;
- ocultar;
- mover;
- renombrar;
- aplicar un diseño.

Una nueva diapositiva puede insertarse con **Ctrl + M**.

El diseño define la disposición de los marcadores de posición para títulos, texto, imágenes, tablas u otros objetos.

### Duplicar frente a copiar

| Operación | Resultado |
|---|---|
| Duplicar diapositiva | Crea una copia inmediata en la misma presentación |
| Copiar y pegar | Permite colocar la copia en otra posición o presentación |
| Ocultar diapositiva | La conserva, pero no se muestra durante la presentación ordinaria |

> ⚠️ **¡Foco Examen!:** Ocultar una diapositiva no la elimina.

## 5. Patrones, temas y plantillas

El **patrón de diapositivas** controla los elementos comunes de varias diapositivas:

- fondos;
- logotipos;
- posición de títulos y textos;
- pies;
- numeración;
- estilos generales.

Una **plantilla** sirve como punto de partida reutilizable para nuevas presentaciones.

| Concepto | Función |
|---|---|
| Patrón | Controla la apariencia común dentro de una presentación |
| Plantilla | Modelo reutilizable para crear nuevas presentaciones |
| Diseño | Distribución de marcadores en una diapositiva |

> ⚠️ **¡Foco Examen!:** Patrón, plantilla y diseño no son sinónimos.

## 6. Texto, listas y estilos

El texto puede introducirse en marcadores o cuadros de texto. Puede modificarse:

- fuente;
- tamaño;
- color;
- alineación;
- sangría;
- interlineado;
- viñetas;
- numeración.

El uso de estilos permite mantener uniformidad. El formato directo actúa solo sobre el objeto o texto seleccionado.

> ⚠️ **¡Foco Examen!:** Aplicar formato directo no modifica automáticamente el patrón ni el resto de diapositivas.

## 7. Imágenes, formas y objetos

Impress permite insertar:

- imágenes;
- formas;
- líneas y conectores;
- tablas;
- gráficos;
- objetos OLE;
- audio y vídeo.

### Orden y agrupación

Los objetos pueden:

- alinearse;
- distribuirse;
- agruparse;
- desagruparse;
- traerse al frente;
- enviarse al fondo.

| Operación | Efecto |
|---|---|
| Agrupar | Convierte varios objetos en una unidad operativa |
| Desagrupar | Permite volver a editar los objetos por separado |
| Traer al frente | Coloca el objeto por delante de los demás |
| Enviar al fondo | Coloca el objeto detrás |

> ⚠️ **¡Foco Examen!:** Agrupar objetos no los fusiona de forma irreversible.

## 8. Tablas y gráficos

Las tablas organizan información en filas y columnas. Los gráficos representan visualmente datos y pueden estar vinculados a datos editables.

Un gráfico no es una imagen estática por definición: conserva una estructura de datos y propiedades mientras siga siendo un objeto gráfico editable.

> ⚠️ **¡Foco Examen!:** Una captura de un gráfico es una imagen; un gráfico insertado como objeto sigue siendo editable.

## 9. Transiciones y animaciones

La **transición** afecta al cambio entre diapositivas. La **animación** afecta a objetos dentro de una diapositiva.

| Concepto | Se aplica a |
|---|---|
| Transición | Diapositiva completa |
| Animación | Texto, imagen u objeto |

Las transiciones pueden configurarse por clic, automáticamente tras un tiempo o según la configuración de la presentación.

Las animaciones pueden definir entrada, énfasis, salida o trayectoria.

> ⚠️ **¡Foco Examen!:** Transición y animación son conceptos distintos.

## 10. Presentación de diapositivas

La presentación puede iniciarse:

- desde el principio con **F5**;
- desde la diapositiva actual con **Mayús + F5**.

Durante la presentación pueden utilizarse controles para avanzar, retroceder, finalizar o acceder a diapositivas concretas.

### Presentación personalizada

Permite crear un recorrido que incluya solo determinadas diapositivas y en un orden específico, sin borrar las demás.

> ⚠️ **¡Foco Examen!:** Una presentación personalizada no duplica necesariamente el archivo ni elimina diapositivas.

## 11. Notas, comentarios y folletos

- Las **notas** sirven de apoyo al presentador.
- Los **comentarios** permiten observaciones de revisión.
- Los **folletos** permiten imprimir varias diapositivas por página.

| Elemento | Uso principal |
|---|---|
| Notas | Apoyo privado del presentador |
| Comentarios | Revisión colaborativa |
| Folletos | Entrega o impresión resumida |

> ⚠️ **¡Foco Examen!:** Las notas no tienen por qué mostrarse al público durante la proyección.

## 12. Impresión y exportación

Impress permite imprimir:

- diapositivas;
- notas;
- esquema;
- folletos.

También puede exportar a PDF. Exportar no sustituye necesariamente el archivo editable ODP.

### Formatos principales

| Formato | Uso |
|---|---|
| ODP | Formato nativo editable |
| PPT/PPTX | Compatibilidad con PowerPoint |
| PDF | Distribución o impresión |

> ⚠️ **¡Foco Examen!:** PDF es adecuado para distribución, pero no conserva la edición normal de la presentación como ODP.

## 13. Atajos esenciales

| Acción | Atajo |
|---|---:|
| Nueva presentación | **Ctrl + N** |
| Guardar | **Ctrl + S** |
| Guardar como | **Ctrl + Mayús + S** |
| Nueva diapositiva | **Ctrl + M** |
| Iniciar desde el principio | **F5** |
| Iniciar desde diapositiva actual | **Mayús + F5** |
| Duplicar objeto o selección | **Ctrl + arrastrar**, según contexto |
| Seleccionar todo | **Ctrl + A** |
| Agrupar | **Ctrl + Mayús + G** |
| Desagrupar | **Ctrl + Alt + Mayús + G** |

> ⚠️ **¡Foco Examen!:** **F5** inicia desde el principio; **Mayús + F5**, desde la diapositiva actual.

## 14. Buenas prácticas

Una presentación eficaz debe:

- mantener coherencia visual;
- limitar texto excesivo;
- utilizar contraste suficiente;
- emplear animaciones con moderación;
- comprobar enlaces y elementos multimedia;
- ensayar la proyección;
- conservar una copia editable.

## 15. Esquema final

```text
IMPRESS
├─ Presentación → archivo ODP
├─ Diapositivas → contenido y orden
├─ Diseños → distribución de marcadores
├─ Patrón → apariencia común
├─ Plantilla → modelo reutilizable
├─ Transición → entre diapositivas
├─ Animación → sobre objetos
├─ Notas → apoyo al presentador
├─ Folletos → varias diapositivas por página
└─ F5 / Mayús+F5 → inicio de presentación
```

> ⚠️ **¡Foco Examen!:** Las asociaciones esenciales son **ODP = formato nativo**, **Ctrl + M = nueva diapositiva**, **F5 = desde el principio**, **Mayús + F5 = desde la actual**, **transición = diapositiva** y **animación = objeto**.
