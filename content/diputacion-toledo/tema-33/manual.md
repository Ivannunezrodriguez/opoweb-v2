# Tema 33 · Microsoft Edge y Micro Focus GroupWise 24

## Epígrafe oficial

**Microsoft Windows 11 Pro II. Navegación por Internet con Microsoft Edge. Uso del correo electrónico con Micro Focus GroupWise 24 y posteriores.**

**Estado editorial:** `GENERACION_DESDE_CERO`

## 1. Microsoft Edge: concepto y elementos básicos

Microsoft Edge es el navegador web de Microsoft integrado en Windows 11. Permite acceder a sitios web, aplicaciones web, descargas, favoritos, historial y contenidos sincronizados.

Elementos principales:

- barra de direcciones y búsqueda;
- pestañas;
- botones Atrás, Adelante y Actualizar;
- menú **Configuración y más**;
- panel de favoritos;
- historial;
- descargas;
- perfiles;
- extensiones.

> ⚠️ **¡Foco Examen!:** La barra de direcciones también funciona como cuadro de búsqueda. No es necesario abrir previamente un buscador.

## 2. Navegación mediante pestañas

Las pestañas permiten mantener varias páginas abiertas dentro de una misma ventana.

| Acción | Atajo |
|---|---:|
| Nueva pestaña | **Ctrl + T** |
| Cerrar pestaña activa | **Ctrl + W** |
| Reabrir pestaña cerrada | **Ctrl + Mayús + T** |
| Ir a la pestaña siguiente | **Ctrl + Tab** |
| Ir a la pestaña anterior | **Ctrl + Mayús + Tab** |
| Nueva ventana | **Ctrl + N** |
| Nueva ventana InPrivate | **Ctrl + Mayús + N** |

> ⚠️ **¡Foco Examen!:** **Ctrl + W** cierra la pestaña actual; **Alt + F4** cierra la ventana activa.

## 3. Favoritos

Los favoritos permiten guardar direcciones web para acceder a ellas posteriormente.

Operaciones habituales:

- añadir la página actual a favoritos;
- crear carpetas de favoritos;
- organizar y ordenar favoritos;
- mostrar u ocultar la barra de favoritos;
- importar y exportar favoritos;
- sincronizarlos mediante el perfil, cuando la organización lo permita.

Atajos útiles:

| Acción | Atajo |
|---|---:|
| Añadir página actual a favoritos | **Ctrl + D** |
| Abrir administración de favoritos | **Ctrl + Mayús + O** |

> ⚠️ **¡Foco Examen!:** Guardar una página en favoritos no guarda necesariamente una copia local de su contenido; conserva principalmente su dirección.

## 4. Historial

El historial registra las páginas visitadas, salvo las limitaciones derivadas del modo InPrivate, la configuración y las políticas corporativas.

Puede utilizarse para:

- recuperar páginas visitadas;
- buscar visitas anteriores;
- eliminar elementos concretos;
- borrar datos de navegación.

| Acción | Atajo |
|---|---:|
| Abrir historial | **Ctrl + H** |
| Borrar datos de navegación | **Ctrl + Mayús + Supr** |

Los datos que pueden borrarse incluyen historial, cookies, archivos en caché, datos de formularios y otros elementos configurables.

> ⚠️ **¡Foco Examen!:** Borrar el historial no elimina necesariamente los archivos descargados; puede eliminar la lista de descargas, pero no los archivos ya guardados.

## 5. Descargas

Edge mantiene un panel con los archivos descargados y su estado.

| Acción | Atajo |
|---|---:|
| Abrir descargas | **Ctrl + J** |

Desde el panel de descargas puede:

- abrir el archivo;
- mostrarlo en su carpeta;
- volver a intentar una descarga;
- cancelar una descarga;
- eliminar la entrada de la lista.

Microsoft Defender SmartScreen puede analizar sitios y descargas para advertir sobre posibles riesgos.

> ⚠️ **¡Foco Examen!:** Eliminar una entrada de la lista de descargas no implica necesariamente borrar el archivo del disco.

## 6. Navegación InPrivate

La navegación InPrivate reduce la conservación local de determinados datos al cerrar todas las ventanas InPrivate.

Al finalizar la sesión se eliminan normalmente:

- historial de navegación;
- cookies y datos de sitios de la sesión;
- datos introducidos en formularios;
- determinados permisos temporales.

No impide necesariamente que la actividad sea visible para:

- la organización;
- el proveedor de acceso;
- el administrador de red;
- los sitios web visitados.

Los favoritos creados y los archivos descargados pueden permanecer después de cerrar la sesión InPrivate.

> ⚠️ **¡Foco Examen!:** InPrivate no equivale a anonimato total ni elimina los archivos descargados.

## 7. Seguridad, permisos y privacidad

Edge permite gestionar:

- cookies;
- permisos de cámara, micrófono y ubicación;
- ventanas emergentes;
- notificaciones;
- contraseñas;
- seguimiento;
- extensiones;
- SmartScreen.

Las extensiones amplían funciones del navegador, pero pueden requerir permisos para leer o modificar datos de determinados sitios.

> ⚠️ **¡Foco Examen!:** Una extensión instalada no debe considerarse automáticamente segura; hay que revisar origen, permisos y política corporativa.

## 8. Buscar, imprimir y guardar páginas

| Acción | Atajo |
|---|---:|
| Buscar texto en la página | **Ctrl + F** |
| Imprimir | **Ctrl + P** |
| Guardar página | **Ctrl + S** |
| Actualizar | **F5** o **Ctrl + R** |
| Forzar recarga | **Ctrl + F5** |

La impresión puede dirigirse a una impresora física o a una impresora virtual, como PDF, según la configuración disponible.

## 9. GroupWise 24: concepto

OpenText GroupWise es una plataforma corporativa de colaboración que integra:

- correo electrónico;
- calendario;
- citas y reuniones;
- tareas;
- notas;
- contactos;
- libretas de direcciones;
- carpetas compartidas;
- gestión de adjuntos.

Puede utilizarse mediante cliente de escritorio y, según la organización, mediante GroupWise Web.

> ⚠️ **¡Foco Examen!:** GroupWise no es solo correo; también integra calendario, tareas, contactos y recursos compartidos.

## 10. Elementos principales de GroupWise

La interfaz habitual incluye:

- lista de carpetas;
- buzón;
- elementos enviados;
- calendario;
- contactos;
- búsqueda;
- panel de lectura;
- barra de herramientas.

Carpetas frecuentes:

| Carpeta | Función |
|---|---|
| Buzón | Elementos recibidos |
| Elementos enviados | Mensajes y citas enviados |
| Trabajo en curso | Borradores y elementos no finalizados |
| Papelera | Elementos eliminados pendientes de purga |
| Archivo | Elementos archivados, si está configurado |

## 11. Redacción y envío de mensajes

Un mensaje puede contener:

- destinatarios **Para**;
- destinatarios **CC**;
- destinatarios **CCO**;
- asunto;
- cuerpo;
- adjuntos;
- prioridad o clasificación, según configuración.

| Campo | Efecto |
|---|---|
| Para | Destinatario principal |
| CC | Copia visible para los demás destinatarios |
| CCO | Copia oculta para los demás destinatarios |

> ⚠️ **¡Foco Examen!:** Los destinatarios incluidos en **CCO** no son visibles para los demás receptores.

## 12. Responder, responder a todos y reenviar

| Acción | Resultado |
|---|---|
| Responder | Contesta al remitente |
| Responder a todos | Contesta al remitente y a los destinatarios incluidos |
| Reenviar | Envía el mensaje a nuevos destinatarios |

Debe utilizarse **Responder a todos** solo cuando la respuesta sea pertinente para todo el grupo.

> ⚠️ **¡Foco Examen!:** Reenviar crea un nuevo envío a otros destinatarios; no es una respuesta dentro del mismo intercambio.

## 13. Adjuntos

Los archivos pueden añadirse como adjuntos al mensaje. El usuario debe comprobar:

- tamaño;
- extensión;
- contenido;
- destinatarios;
- restricciones de seguridad;
- ausencia de datos innecesarios.

No debe abrirse un adjunto sospechoso aunque proceda aparentemente de un contacto conocido.

## 14. Calendario, citas y reuniones

GroupWise permite crear:

- citas personales;
- reuniones con asistentes;
- eventos de día completo;
- recordatorios;
- tareas.

Una reunión enviada a otros usuarios puede generar respuestas de aceptación, rechazo o propuesta alternativa, según configuración.

| Elemento | Rasgo principal |
|---|---|
| Cita personal | Solo afecta al calendario propio |
| Reunión | Se envía a otros asistentes |
| Tarea | Incluye seguimiento y posible fecha de vencimiento |
| Nota | Recordatorio informativo |

> ⚠️ **¡Foco Examen!:** Una cita personal no invita automáticamente a otros usuarios; una reunión sí se distribuye a los asistentes.

## 15. Contactos y libretas de direcciones

Las libretas de direcciones permiten:

- buscar usuarios;
- crear contactos personales;
- crear grupos;
- seleccionar destinatarios;
- consultar datos de contacto;
- utilizar libretas compartidas, si existen permisos.

Un grupo de contactos facilita el envío a varios destinatarios, pero no crea necesariamente una carpeta compartida ni un calendario común.

## 16. Carpetas, categorías y organización

Los mensajes pueden organizarse mediante:

- carpetas personales;
- carpetas compartidas;
- categorías;
- reglas;
- prioridades;
- búsquedas.

Las reglas pueden automatizar acciones como mover, reenviar, responder o clasificar elementos, según los permisos y políticas establecidas.

> ⚠️ **¡Foco Examen!:** Una carpeta compartida requiere permisos; crear una carpeta personal no la hace visible para otros usuarios.

## 17. Eliminación, Papelera y purga

Al eliminar un elemento puede trasladarse a la Papelera. Mientras no se purgue, puede ser recuperable.

No debe confundirse:

| Acción | Efecto |
|---|---|
| Eliminar | Traslada normalmente a Papelera |
| Restaurar | Devuelve el elemento a una carpeta |
| Purgar o vaciar | Elimina definitivamente según configuración |

> ⚠️ **¡Foco Examen!:** Eliminar y purgar no son equivalentes.

## 18. Seguridad en el correo

Buenas prácticas:

- verificar remitente y dominio;
- desconfiar de enlaces urgentes o inesperados;
- no facilitar credenciales;
- revisar adjuntos;
- comprobar destinatarios antes de enviar;
- evitar datos sensibles innecesarios;
- utilizar CCO cuando proceda para proteger direcciones;
- respetar las normas de archivo y confidencialidad.
