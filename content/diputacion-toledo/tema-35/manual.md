# Tema 35. LibreOffice 24 I. Elaboración y uso de documentos de texto con LibreOffice Writer

## 1. Writer y el documento de texto

LibreOffice Writer es el procesador de textos de la suite LibreOffice. Permite crear, editar, dar formato, revisar, imprimir y exportar documentos sencillos y complejos.

El formato nativo es **ODT**. También puede abrir y guardar en formatos como DOCX, RTF y TXT, aunque determinadas funciones pueden no conservarse idénticamente al cambiar de formato.

> ⚠️ **¡Foco Examen!:** Guardar en DOCX no convierte Writer en Word ni garantiza compatibilidad perfecta; el formato nativo de Writer es **ODT**.

## 2. Interfaz de Writer

La ventana de Writer incluye:

- barra de título;
- barra de menús;
- barras de herramientas;
- área de edición;
- reglas horizontal y vertical;
- barra lateral;
- barra de estado;
- controles de zoom.

La barra de estado informa, entre otros datos, de página, número de palabras, idioma, estilo de página, modo de inserción y escala de visualización.

## 3. Crear, abrir, guardar y cerrar

| Acción | Atajo |
|---|---:|
| Nuevo documento | **Ctrl + N** |
| Abrir | **Ctrl + O** |
| Guardar | **Ctrl + S** |
| Guardar como | **Ctrl + Mayús + S** |
| Cerrar documento | **Ctrl + W** |
| Imprimir | **Ctrl + P** |

**Guardar** actualiza el archivo actual. **Guardar como** permite modificar nombre, ubicación o formato.

> ⚠️ **¡Foco Examen!:** Guardar como puede crear un archivo distinto; guardar actualiza el documento actual.

## 4. Selección, edición y portapapeles

Las operaciones esenciales son:

- seleccionar texto con ratón o teclado;
- copiar con **Ctrl + C**;
- cortar con **Ctrl + X**;
- pegar con **Ctrl + V**;
- deshacer con **Ctrl + Z**;
- rehacer con **Ctrl + Y**;
- seleccionar todo con **Ctrl + A**.

Pegar sin formato evita arrastrar estilos ajenos. Puede utilizarse desde Pegado especial.

| Operación | Efecto |
|---|---|
| Copiar | Duplica sin eliminar el original |
| Cortar | Prepara el contenido para moverlo |
| Pegar | Inserta el contenido del portapapeles |
| Pegar sin formato | Inserta texto sin formato de origen |

## 5. Formato de caracteres

El formato directo afecta al texto seleccionado:

- tipo y tamaño de letra;
- negrita, cursiva y subrayado;
- color de fuente y resaltado;
- superíndice y subíndice;
- efectos tipográficos.

Atajos frecuentes:

| Formato | Atajo |
|---|---:|
| Negrita | **Ctrl + B** |
| Cursiva | **Ctrl + I** |
| Subrayado | **Ctrl + U** |

El formato directo y los estilos no son equivalentes. Los estilos permiten homogeneidad y cambios globales.

> ⚠️ **¡Foco Examen!:** Un estilo modifica de forma coherente todos los elementos que lo usan; el formato directo actúa solo sobre la selección concreta.

## 6. Formato de párrafo

Las propiedades de párrafo comprenden:

- alineación izquierda, centrada, derecha y justificada;
- sangrías;
- espaciado anterior y posterior;
- interlineado;
- tabulaciones;
- bordes y fondo;
- control de líneas viudas y huérfanas.

| Alineación | Atajo |
|---|---:|
| Izquierda | **Ctrl + L** |
| Centrada | **Ctrl + E** |
| Derecha | **Ctrl + R** |
| Justificada | **Ctrl + J** |

Las marcas no imprimibles pueden visualizarse con **Ctrl + F10** y ayudan a detectar espacios, tabulaciones, saltos y finales de párrafo.

## 7. Estilos y plantillas

Writer utiliza estilos de:

- párrafo;
- carácter;
- página;
- marco;
- lista;
- tabla.

Los estilos de página controlan orientación, tamaño, márgenes, encabezados, pies y columnas. Las plantillas contienen configuraciones y elementos reutilizables para nuevos documentos.

| Concepto | Función |
|---|---|
| Estilo | Conjunto reutilizable de propiedades de formato |
| Plantilla | Documento modelo con estilos, estructura y elementos predefinidos |

> ⚠️ **¡Foco Examen!:** La orientación y los márgenes se gestionan mediante estilo de página, no mediante estilo de carácter.

## 8. Configuración de página

Writer permite configurar:

- tamaño de papel;
- orientación vertical u horizontal;
- márgenes;
- encabezados y pies;
- numeración de páginas;
- columnas;
- fondo y bordes.

Un salto de página fuerza el inicio en una nueva página. No debe sustituirse por múltiples retornos de carro.

> ⚠️ **¡Foco Examen!:** Pulsar Intro repetidamente no equivale a insertar un salto de página.

## 9. Listas

Writer admite listas:

- con viñetas;
- numeradas;
- multinivel.

La numeración automática mantiene la secuencia y permite reiniciar o continuar numeración. Los estilos de lista proporcionan consistencia.

## 10. Tablas

Las tablas organizan información en filas y columnas. Pueden insertarse desde el menú Tabla o mediante **Ctrl + F12**.

Operaciones habituales:

- insertar o eliminar filas y columnas;
- combinar o dividir celdas;
- ajustar ancho y alto;
- aplicar bordes y fondos;
- alinear contenido;
- repetir filas de encabezado;
- ordenar datos.

> ⚠️ **¡Foco Examen!:** Combinar celdas crea una celda a partir de varias; dividir celdas realiza la operación inversa.

## 11. Imágenes y objetos

Las imágenes pueden insertarse, redimensionarse, recortarse y anclarse. El ajuste determina cómo fluye el texto alrededor del objeto.

Tipos de anclaje habituales:

- a la página;
- al párrafo;
- al carácter;
- como carácter.

El anclaje y el ajuste son conceptos distintos: el primero determina la referencia del objeto; el segundo, la relación visual con el texto.

## 12. Encabezados, pies y numeración

Los encabezados y pies pertenecen al estilo de página. Pueden contener texto, fecha, título o número de página.

Para numeración automática debe insertarse el campo **Número de página**, no escribir manualmente el número.

> ⚠️ **¡Foco Examen!:** Escribir “1” en el pie no genera numeración automática; debe insertarse el campo correspondiente.

## 13. Búsqueda y sustitución

- Buscar: **Ctrl + F**.
- Buscar y reemplazar: **Ctrl + H**.

La sustitución puede aplicarse individualmente o a todas las coincidencias. También admite opciones como coincidencia de mayúsculas, palabras completas y expresiones regulares.

## 14. Revisión del documento

Writer permite:

- corrección ortográfica;
- diccionarios e idioma del texto;
- sinónimos;
- comentarios;
- control de cambios;
- comparación de documentos.

La revisión ortográfica depende del idioma asignado al texto y de que exista diccionario instalado.

El control de cambios registra inserciones, eliminaciones y modificaciones, que después pueden aceptarse o rechazarse.

## 15. Campos, índices y combinación de correspondencia

Los campos muestran información variable como fecha, autor o número de página.

Los índices y sumarios automáticos se generan a partir de estilos de título. Si los títulos se formatean manualmente sin estilos, el sumario puede no reconocerlos correctamente.

La combinación de correspondencia integra:

- un documento principal;
- una fuente de datos;
- campos de combinación;
- documentos resultantes personalizados.

> ⚠️ **¡Foco Examen!:** El sumario automático depende de la estructura de estilos de título, no solo del tamaño visual de la letra.

## 16. Impresión y exportación

Writer permite vista previa, selección de páginas, copias y configuración de impresora. Puede exportar directamente a PDF.

Exportar a PDF crea un archivo para distribución; no sustituye necesariamente al documento editable original.

| Acción | Resultado |
|---|---|
| Guardar como ODT | Conserva el documento editable nativo |
| Exportar a PDF | Genera un archivo de distribución y lectura |
| Imprimir | Envía el trabajo a una impresora física o virtual |
